Ruby on Rails application: uses [Edamam Api](https://developer.edamam.com/edamam-recipe-api) to search and display available recipes.

* Ruby version: 2.4.0
* Rails version: 5.1.4
* Foundation version: 6.4.1.2

* Run locally:
  + rails server
  + http://localhost:3000

* How to run the test suite:
  rails test
